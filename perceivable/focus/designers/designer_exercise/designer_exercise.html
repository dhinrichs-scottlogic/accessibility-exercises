Visual Focus - Exercise

Visual Focus is a cue to indicate for the user which part of the application is currently active. Now it's time to apply what you learned in the introduction section. Try to first fix it on your own, and then compare your code with the solution.

Tasks

MVP

Write instructions for the developers to change the form so that each interactive field has a visual focus. Provide a wireframe, or a video of an example of good keyboard focus that shows developers how the end result should work & look like.

  • Find a good example of visual focus
  • Create a screenshot, video, or wireframe of what the UI should look like
  • Write a short summary of the main design considerations
  • Write a bullet point list of your design specifications
  • Check your result with the suggested solution
