Keyboard focus

Try to navigate through it using your keyboard. You can use the following controls:

    • Tab: Move to the next interactive element
    • Shift + Tab:Move to the previous interactive element
    • Arrow keys: Navigate through radio buttons and checkboxes
    • Space: Activate the currently focused element
    • Enter: Submit the form
  • Ensure the form so that it can be navigated using only a keyboard
